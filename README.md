# Network Rules

Единый репозиторий доменных правил для роутера OpenWrt/GL.iNet, Shadowrocket и будущих VPN-конфигураций.

## Структура

- `domains/proxy/` — сервисы, которые нужно отправлять через VPN.
- `domains/direct/` — сервисы, которые лучше оставлять напрямую.
- `openwrt/` — скрипты для GL.iNet/OpenWrt.
